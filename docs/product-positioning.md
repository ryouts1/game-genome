# Product positioning

## 何として見せるか

`game-genome` は「ゲームを 1 本作る repo」ではなく、**JSON 定義から複数の盤面ゲームを試作・比較・共有し、さらに学習済みアセットまで配布できる基盤**として見せるのが自然です。

## 想定する使い方

1. 既存サンプルを開く
2. Studio でルールを少し変える
3. 検証と health check を見る
4. Minimax / MCTS / Genome AI で局面分析する
5. Arena でバッチ比較する
6. 良ければ JSON と学習済みブックをまとめて派生 repo に切り出す

## 価値の伝え方

- 実装力: UI / engine / AI / learning / tooling を 1 本で通している
- 設計力: JSON DSL と学習済みアセットの責務を分けている
- 継続力: サンプルゲームと学習済みブックを増やしやすい構造にしている
- 説明力: README、catalog report、training report で入口を揃えている

## 売り方を寄せるなら

受託や個人販売に寄せるなら、次の見せ方が合います。

- 教材用: ルール変更と AI の差を見せる授業・研修用基盤
- プロトタイピング用: 同人・研究・授業向けの盤面ゲーム試作基盤
- 検証ツール用: ルール変更と自己対戦を短いサイクルで回す内部ツール
- 配布用キット: サンプルゲームと学習済みブックを同梱した starter kit
